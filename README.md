# Калькулятор с историей

## 📝 Задание
Реализуйте объект `calculator` с методами для арифметических операций и ведением истории.

## ✅ Требования
- Все методы должны валидировать входные данные (числа)
- Деление на ноль должно выбрасывать ошибку
- История должна хранить все операции
- Метод `getHistory()` возвращает копию истории

## 📚 API
- `add(a, b)` - сложение
- `subtract(a, b)` - вычитание
- `multiply(a, b)` - умножение
- `divide(a, b)` - деление
- `getHistory()` - получить историю
- `clearHistory()` - очистить историю
