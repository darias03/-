# алькулятор с историей

![Tests](https://github.com/darias03/-/actions/workflows/test-and-push.yml/badge.svg)

## 📝 адание
еализуйте объект calculator с методами для арифметических операций и ведением истории.

## 📚 API
- dd(a, b) - сложение
- subtract(a, b) - вычитание
- multiply(a, b) - умножение
- divide(a, b) - деление
- getHistory() - получить историю
- clearHistory() - очистить историю
